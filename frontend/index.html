Astra RGB-D + OpenPose Edition

ICH Dance Motion Digitization System

Local web console for low-cost intangible cultural heritage motion capture, 3D reconstruction, optimization, and Blender export.

Device Test

Check Astra RGB-D camera access and generate mock RGB/depth test frames.

Run BODY_25 pose analysis. V0.1 writes example keypoint JSON files.

Step 4

Reconstruct 3D Coordinates

Fuse 2D keypoints with depth values to produce x, y, z coordinates.

Step 5

Optimize Motion Data

Smooth temporal jitter and prepare a clean 3D motion CSV.

Step 6

Export CSV / BVH / Blender

Export research data and a simplified BVH motion file for Blender import.
